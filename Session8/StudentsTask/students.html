JSON Questions:

4a. Can JSON hold null and Boolean values?
Yes, JSON supports: strings, numbers, booleans, null, arrays, and objects.

4b. What are the differences between XML and JSON?
1. Syntax: XML uses tags, JSON uses key-value pairs
2. Readability: JSON is more concise and human-readable
3. Data Types: JSON has built-in types; XML stores everything as text
4. Parsing: JSON parsing is faster and simpler
5. Size: JSON files are typically smaller
6. Structure: XML supports attributes and namespaces; JSON does not
